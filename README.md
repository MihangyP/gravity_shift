# 🚀 Gravity Shift 

Gravity Shift is a minimalist 2D game where the player controls a square that can invert gravity to avoid obstacles and survive as long as possible.

> **Note:** This game is still in development.

## 🎮 Concept
You control a square moving through an environment filled with obstacles.
The core mechanic is simple but powerful: gravity inversion.
- Normal graviy -> the player falls downward
- Inverted gravity -> the player falls upward
The challenge is to **master timing and reflexes** to avoid obstacles and progress.

## ✨ Features
- 🎯 Simple and addictive gameplay
- 🔄 Real-time gravity switching
- ⚙️  Basic physics system
- 🧠 Reflex and logic-based gameplay

## 🛠 Technologies
- Language: C
- Graphics: Raylib

## 🎯 Project Goals
This project aims to:
- Learn the fundamentals of **game development**
- Implement a simple physics system (gravity)
- Handle collision detection
- Build a clean and efficient **game loop**
- Experiment with gameplay mechanics

## 🎮 Controls

| Key   | Action         |
| ----- | -------------- |
| ⬅ / ➡ | Move           |
| Space | Invert gravity |
| ESC   | Quit           |

## 📦 Installation
> **Note:** For now, the game is only available on Linux
```bash
git clone https://github.com/MihangyP/gravity_shift.git
cd gravity_shift
make
./gravity_shift
```
